### **Your memory shouldn't end where your attention does.**

**Digitalism** is a **digital mind extension** that continuously understands your work, conversations, and activity to create a searchable, contextual memory layer for your life.

It quietly observes your workflow, connects information across applications, remembers what matters, and gives you an AI companion that understands **what you've already seen, heard, and done.**

# 🧠 What is Digitalism?

Digitalism is a **personal cognitive operating system**.

Rather than acting as another chatbot waiting for prompts, Digitalism continuously builds a structured understanding of your digital life.

It becomes an extension of your own memory.

Imagine having an AI that remembers:

- everything you've worked on
- every meeting
- every idea
- every document
- every conversation
- every research session
- every browser rabbit hole

...without requiring you to organize any of it manually.

---

# ✦ Core Philosophy ✦

Digitalism is built around a simple belief:

> **Your computer already knows your workflow. Your AI should too.**

Instead of isolated conversations, Digitalism creates persistent understanding that grows alongside you.

Your digital history becomes searchable knowledge instead of forgotten context.

# 🚀 Quick Start

Clone the repository.

```bash
git clone https://github.com/OleLehmann/Digitalism.git

cd Digitalism
```

Install dependencies.

```bash
npm install
```

Run setup.

```bash
npm run setup
```

Launch.

```bash
npm run dev
```
